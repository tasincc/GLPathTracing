GLPathTracing 
=================

基于GLSL的pathTracing Rendering，这是一个个人学习使用的工程项目，实现一个GPU加速的光线追踪渲染器，目的是为加深对于渲染本质的理解。在这个项目中我希望不仅仅是基本的实现
pathTracing的算法，我希望它能够有一整套完善的框架结果，包括场景描述文件、Gizmo、Inspector GUI等功能，实现过程会大量参考其他的优秀项目来逐步完善它。
